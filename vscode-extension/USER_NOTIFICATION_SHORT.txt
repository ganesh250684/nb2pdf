**Why this happens:**
The extension needs a Python library called "reportlab" to create PDFs, but it wasn't installed on your system.

**Good news!** I've already published an updated version (v1.1.0) that:
- Checks for reportlab BEFORE converting
- Shows a clear error message
- Offers an "Install Now" button
- Provides step-by-step installation guide

**To get the update:**
Open Extensions in VS Code ‚Üí Search "nb2pdf" ‚Üí Click "Update"

**Still not working?**

```

```
